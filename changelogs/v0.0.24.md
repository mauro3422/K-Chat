## New Features

### Context-aware loop detector
- `loop_detector.py`: `_inside_code_block()` now counts ANY triple backtick (not just `html-widget`) — phrase repetition check skips inside ALL code blocks
- `loop_detector.py`: token check (same word 15×) still runs everywhere — repeating the same word is always a loop

### Transparent stream retry
- New `stream_retry_handler.py`: decoupled module that builds continuation context (partial output + "continue from here") and calls the LLM again transparently
- `chat_stream.py`: on loop detection, first attempts transparent recovery via `StreamRetryHandler` — if the model produces new content, the stream continues seamlessly; only falls back to `loop_detected` error if recovery fails
- Configurable `max_retries` (default 1), can be disabled by passing `retry_handler=None`
