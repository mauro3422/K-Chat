# Release v0.0.43

- **Decoupled Rate-Limit Retry Logic**: Isolated exponential backoff retry logic into `src/llm/retry.py` and central config `src/constants.py` for generic reuse.
- **Optimized Unit Tests**: Patched `time.sleep` inside mock LLM test suites to reduce test runtimes from 43s to 12s.
- **Truncation-Safe Loop Detection**: Switched `LoopDetector` to track triple backticks via an absolute counter delta, avoiding false loop triggers when long streams get truncated.
- **Widget Mode Suspension**: Suspended phrase repetition loop checking during widget generation to allow CSS/layout repetition.
- **Code Block Widget Exclusion**: Aligned client-side detectors (`core.js` and `widget-detector.js`) and backend message parser (`widget_contract.py`) to ignore widget triggers nested inside standard markdown code blocks and inline code backticks.
- **Vite/JS Widget Rendering Fix**: Fixed index offset shifting bug in `widget-container-renderer.js` when removing parsed widget text.
- **CPU/TDP Log Optimization**: Disabled high-frequency console/POST logs inside streaming loop to preserve CPU cycles on SUMA C10.
