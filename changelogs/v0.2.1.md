# v0.2.1 - 2026-06-29

## Corregido

- Cierre asincronico del runtime compartido en tests y shutdown web para evitar leaks de `aiosqlite` y estados cruzados entre casos.
- Repositorios de memoria migrados a manejo explicito de conexiones con DI, cerrando conexiones propias sin cerrar conexiones inyectadas.
- Cierre explicito de conexiones SQLite sincronas en `MemoryWorkCatalogRepository` y `track_retrieval`, incluyendo rutas de error.
- `HybridRetriever` ahora usa los resultados normalizados cuando una busqueda parcial falla, manteniendo la degradacion no fatal.
- Registro local de modelo `fastembed` para evitar warnings por `FlagEmbedding` y preservar el modelo multilingue CLS.
- Tests de cache/vectorizacion actualizados con fixtures de teardown para cerrar conexiones en memoria.

## Agregado

- Preflight de pipeline de memoria para backfill conservador, auditoria y comparacion entre nodos.
- Helper de routers web para resolver repositorios desde request/app state sin acoplar dominio a FastAPI.
- Mejoras en control remoto LAN: doctor, memory preflight, pull remoto, hints operativos y cobertura de scripts.
