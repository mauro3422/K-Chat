# v0.0.50 — 2026-06-12

## Cambios

- `validate_javascript` now writes a temporary file and runs `node --check` on it, which restores the JS validation path on current Node versions.
- `list_files` and `search_files` now support simple brace globs like `*.{py,js}` in their file filters.
- `build_summary()` now handles both AST-style dict entries and regex-style string entries, so mixed-language directory listings do not break on JS/TS files.
- Added regression tests for JS validation and brace glob matching in the new tools.
