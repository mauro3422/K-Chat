# Release v0.0.46

- **New tools oficializadas**: `search_files` y `edit_file` se sumaron al registry como tools de primera clase, con documentación y tests.
- **Hardening de filesystem**: `search_files` y `edit_file` quedaron alineadas con la política de paths seguros del resto de tools.
- **Helpers compartidos**: se introdujeron `_analyzers.py` y `_validators.py` para reducir duplicación entre tools de análisis y edición.
- **Docs y tests alineados**: README, resúmenes de arquitectura y pruebas del loader quedaron actualizados al registry real.
