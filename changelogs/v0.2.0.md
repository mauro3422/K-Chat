# v0.2.0 — 2026-06-26

## Agregado

- **Codex Task Bridge**: Nuevo sistema de delegación LAN. Servicio `codex_task_bridge.py` + router `codex.py` + tool `delegate_to_codex.py` — permite delegar tareas de un agente a otro en la red LAN, con contexto de delegación en mensajes remotos.
- **LAN Remote Control**: Sistema completo de control remoto con smoke tests (`scripts/lan_field_smoke.py`), nodos configurables (`ops/remote/nodes.example.json`), script PowerShell (`scripts/remote-kairos.ps1`) y cliente remoto (`ops/remote/kairos_remote.py`). Incluye failover de modelos sobre LAN.
- **Descubrimiento LAN dinámico**: Los nodos se descubren sin necesidad de IPs fijas mediante `lan_discovery.py`. Bind automático a la interfaz activa y aprendizaje recíproco de peers desde heartbeats.
- **Contexto de delegación Codex**: Los mensajes remotos incluyen contexto de delegación para que el agente destino entienda el origen de la solicitud.
