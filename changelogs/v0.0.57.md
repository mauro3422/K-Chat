# v0.0.57 — Lego Architecture: Desacoplamiento Profundo

## Cambios Arquitectónicos

### API Layer — Framework-agnostic
- `src/api/exceptions.py` — Nueva `ServiceException` (sin dependencia FastAPI)
- `src/api/session.py` — `HTTPException` → `ServiceException`
- `web/app_factory.py` — Único punto de traducción `ServiceException → JSONResponse`

### Memory — Protocolos limpios
- `src/memory/types.py` — Nuevo módulo compartido con `MessageRecord`
- `src/memory/repos/protocols.py` — Ya no importa desde la implementación
- `src/memory/repos/message_repository.py` — Importa desde `types.py`

### Tools — Sin dependencia a Core
- `src/tools/_contract.py` — Nuevo: `HistoryMessage` compartido
- `src/core/history_contract.py` — Re-exporta desde `_contract` (backward compat)
- `src/tools/runner.py` — Refactor: `_prepare_rate_limiters()`, `_yield_tool_error_events()`, `_execute_and_persist_tools()`
- `src/tools/_tool_persister.py` — Importa desde `_contract` en vez de `core`

### Config — Dependency Injection
- `src/tools/web_search.py` — Acepta `config` param
- `src/tools/fetch_url.py` — Acepta `config` param
- `src/llm/providers.py`, `retry.py`, `discovery.py` — Imports lazy + `config` param
- `src/constants.py` — `MAX_TOOL_TURNS` → función `max_tool_turns(config=None)`
- `src/context/templates.py` — `TEMPLATES` → `get_templates(config=None)`
- `src/gateway.py` — Lazy import + config explícito

### Orchestrator — Split de god dataclass
- `src/core/orchestrator_contract.py` — `OrchestratorDeps` dividido en 4 sub-dataclasses: `LLMDeps`, `ToolDeps`, `StorageDeps`, `RequestStateDeps`. `OrchestratorDeps` mantiene compat como facade.
- `src/core/orchestrator.py` — `chat_stream()` acepta sub-grupos como kwargs opcionales

### Tool Loop — Unificación dual paths
- `src/core/tool_loop.py` — Extraídos `_build_ctx()`, `_set_debug()`, `_run_tool_loop()`. `run_tool_loop_sync()` ahora delega en `_run_tool_loop()`.

### Tests — Shared fixtures
- `tests/unit/conftest.py` — Nuevos fixtures: `in_memory_db`, `repos`, `config`, `tool_registry`, `history_service`
- `tests/unit/test_runner.py` — 8 tests migrados a fixtures reales
- `tests/unit/test_core.py` — 9 tests reescritos con async generators
- `tests/unit/test_tool_loop_sync.py` — 6 tests reescritos

### Documentación actualizada
- `docs/HEALTH.md`, `docs/MODULES.md`, `docs/llm_architecture.md`, `docs/tools-architecture.md`, `docs/ARCHITECTURE_FRONTEND.md`

### Frontend bugfix
